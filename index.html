Pet Cloud
Github Repository Deployed with Heroku

This project is focused on giving a medical history for our pets in Chile, where every checkup, vaccine, or even the diseases they have, are kept on each veterinarian database. The solution we develop, was made in React (Front End) with a Flux structure, using libraries like React-Router, React-Bootstrap, and SweetAlert. On the Back End, Python with Flask and Flask-JWT, SQLAlchemy, and PostgreSQL for our database. We use HL7 FHIR through Hapi FHIR server to follow international health standards. Also we incorporate AWS SES and google places API to give a better user experience. Unfortunately, nowadays we don't have access to Hapi FHIR server and this project is not available anymore.
